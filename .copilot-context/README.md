# Who Killed the Webhook — Copilot Session Context

> 此資料夾保存了 Copilot CLI 的工作上下文，方便在不同機器上繼續討論。
> 推到 Git 後，在公司機器上可以直接告訴 Copilot「請閱讀 .copilot-context/ 下的檔案來恢復上下文」。

## 檔案說明

| 檔案 | 內容 |
|------|------|
| `plan.md` | 完整實作計劃（架構、流程、線索設計） |
| `session-log.md` | 討論記錄（需求確認、決策、問題排查過程） |
| `todo-status.md` | 任務狀態追蹤 |
| `next-steps.md` | 下一步待辦事項 & 已知問題 |
